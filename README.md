## 前言

对于一个优秀的软件架构而言，软件初期的技术选型，开发过程中的系统分割（服务化，SOA，微服务），以及系统上线之后的监控运维，这三方面都是非常重要的。这一篇主要记录对一个线上项目进行监控，通过日志的采集与记录可以有效的观察程序的瓶颈和异常信息，对线上bug的排除有着极其重要的作用。

## 技术选型

- spring
- spring mvc
- redis
- logback
- ELK
